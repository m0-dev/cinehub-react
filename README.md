# cinehub-react
A simple and modern movie discovery app built with React.

## Tech Stack
* React
* JavaScript
* Vite

## Getting started

### Prerequesites

- Node.js v25.2.1
- npm (comes with Node.js)

### Installatiion & Run

Clone the cinehub-react repository from GitHub to your local machine and go into the project folder.
```bash
git clone https://github.com/m0-dev/cinehub-react.git
cd cinehub-react
```

Install all dependencies defined in package.json.
```bash
npm install
```

Start the local development server.
- This launches Vite's dev server
- View the app in your webbrowser `http://localhost:5173`
```bash
npm run dev
```
